Aula 05 - Estrutura de repetição "for"

1. Descreva para que serve o comando "for".

Resposta: 
O comando for é um loop que permite que o programa seja executado um determinado número de vezes.

2. Como o comando "for" define o início de uma repetição?

Resposta:
É passada uma situação inicial.
(let i = 0)

3. Como o comando "for" determina o fim da execução?

Rasposta:
É passada uma condição e uma ação a ser executada a cada repetição. 
(i <= 10; i++)

4. Como ocorre a alteração do valor em cada repetição dentro da estrutura de repetição "for"?

Resposta:
A cada repetição do loop, a condição é testada; se é verdadeira ( true ), o bloco de comandos, e o incremento são executados. 
Quando a condição se torna falsa ( false ), o loop termina.
